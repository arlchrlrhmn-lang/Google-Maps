# Google-Maps Android App
Aplikasi android yang mengimplementasikan Google Maps dengan berbagai jenis tampilan peta untuk meningkatkan pengalaman pengguna dalam menjelajahi lokasi.

# Fitur Utama
- Menampilkan peta menggunakan Google Maps API
- Mengganti tipe peta:
  1.Normal
  2.Satellit
  3.Hybrid
  4.Terrain

# Tech Stack
- Kotlin
- Google Maps API
- Android Studio
